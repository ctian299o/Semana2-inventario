🧾 Sistema de Inventario

Aplicación en Python que permite gestionar productos desde consola de forma sencilla.

---

🚀 Funcionalidades

- Agregar productos con nombre, precio y cantidad
- Validar datos ingresados por el usuario
- Mostrar todos los productos del inventario
- Calcular el valor total y la cantidad de productos
- Menú interactivo fácil de usar

---

🛠️ Tecnologías

- Python

---

▶️ Cómo usar

1. Ejecuta el archivo principal
2. Elige una opción del menú
3. Sigue las instrucciones en pantalla

---

📌 Ejemplo

Producto: Arroz | Precio: 2000 | Cantidad: 3 | Total: 6000
